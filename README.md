# Actividad 3 - Flujo de Control Parte 2

## Descripción

Este programa fue desarrollado en C++ como parte de la Actividad 3 de Lógica de Programación.

El programa solicita la cantidad de estudiantes que se desea registrar. Luego, para cada estudiante, permite ingresar el nombre y cuatro calificaciones. Con esos datos calcula el promedio y determina si el estudiante aprobó o reprobó. Finalmente, muestra toda la información organizada en forma de tabla.

## Objetivo

Aplicar el uso de bucles y estructuras de control para procesar la información de varios estudiantes, calcular su promedio y mostrar el resultado correspondiente.

## Funciones utilizadas

* Variables.
* Entrada y salida de datos.
* Ciclo `for`.
* Condicional `if - else`.
* Cadenas de texto (`string`).
* `setw()` para organizar la información en columnas.
* `setprecision()` para mostrar el promedio con dos decimales.

## Ejemplo de funcionamiento

1. El usuario indica la cantidad de estudiantes.
2. Se ingresan el nombre y las cuatro calificaciones de cada estudiante.
3. El programa calcula el promedio.
4. Se determina si el estudiante aprobó o reprobó.
5. Se muestran los resultados en una tabla.
